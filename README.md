## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ 
- npm, yarn, pnpm, or bun
- Git

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/ddumbaabdallah/genealogy-projects.git
cd genealogy-projects
```

2. **Choose a project to run**
```bash
cd family-story-ui-main  # or family-weaver-ui-main or heritage-hub-ui-main
npm install
npm run dev
```

3. **Open in browser**
```
http://localhost:8080
```
